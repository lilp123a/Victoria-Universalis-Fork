# Victoria Universalis Fork

This is a fork of [Victoria Universalis](https://github.com/vic21984/Victoria-Universalis) version 0.85.

## Changes
- Utility decisions reworked and organised
- National ideas reworked and moved into a decision, to be taken at the player's leisure
- Province selector options reworked
- Added Total Control, a toggleable mode available if the manual economic control gamerule is enabled, which disables capitalists messing with factories so you can better micromanage your economy
- Tech school change decisions reworked to be more flexible
- Various unit types removed (Light artillery, horse artillery, dragoons, aircraft carriers) to simplify military
- Low-impact R&D reforms removed
- Gamerules moved from events into decisions
- New industriousness national value and changes to other national values
- AI can use promotion national focuses
- HFM flag frames
- Various flags ported from GFM
- Localisation cleaned up
- Various other fixes, changes and additions
